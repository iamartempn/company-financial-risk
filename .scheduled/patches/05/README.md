Скоринг финансовой устойчивости компаний по бухгалтерской отчетности. Altman Z-score, LightGBM, SHAP-объяснения.

## Данные

Источник: API bo.nalog.ru (ФНС России). Для каждой компании по ИНН запрашивается бухгалтерская отчетность (выручка, активы, обязательства, EBIT, оборотный капитал, нераспределенная прибыль).

Резервный источник: датасет UCI Polish Companies Bankruptcy - содержит финансовые показатели польских компаний с разметкой факта дефолта. Используется при недоступности API.

## Методология

1. **Altman Z-score** (модификация для непубличных компаний): Z' = 0.717*X1 + 0.847*X2 + 3.107*X3 + 0.420*X4 + 0.998*X5, где X1-X5 - стандартные финансовые коэффициенты. Пороги: Z' < 1.23 - зона дефолта, Z' > 2.9 - безопасная зона.

2. **LightGBM-классификатор**: обучается на наборе финансовых коэффициентов (z_score, debt_ratio, roa, asset_turnover, working_capital_ratio, retained_earnings_ratio). Целевая переменная - факт дефолта (0/1).

3. **SHAP waterfall**: для каждой компании строится диаграмма вклада признаков в итоговый скор, что позволяет объяснить причину присвоенного риск-класса.

## Запуск

Установить зависимости:
```
pip install -r requirements.txt
```

Сбор данных по списку ИНН (файл `data/inns.txt`, один ИНН на строку):
```
python -c "
from src.collector import collect_batch
inns = open('data/inns.txt').read().splitlines()
collect_batch(inns, 'data/raw.duckdb')
"
```

Обучение модели:
```
python -c "
import polars as pl
from src.features import compute_ratios
from src.model import train
df = pl.read_parquet('data/features.parquet')
train(df, 'models/lgbm.pkl')
"
```

Запуск Streamlit-приложения:
```
streamlit run src/app.py
```

## Результат

Пользователь вводит ИНН компании, нажимает "Рассчитать" и получает карточку риска: Z-score, вероятность дефолта, риск-класс (Низкий / Средний / Высокий) и SHAP waterfall с расшифровкой, какие именно показатели тянут компанию в зону риска.

## Интерпретация Z-score Альтмана

| Зона | Z-score | Интерпретация |
|---|---|---|
| Дистресс | Z < 1.23 | Высокая вероятность банкротства в течение 2 лет |
| Серая зона | 1.23 ≤ Z < 2.90 | Неопределённость, требует дополнительного анализа |
| Безопасная зона | Z ≥ 2.90 | Финансово устойчивая компания |

**Применимость к российской отчётности:** оригинальная модель Альтмана калибровалась на данных американских компаний 1960-х годов. Для российских предприятий пороговые значения могут отличаться — особенно в отраслях с высокой долговой нагрузкой (строительство, ретейл). Модель LightGBM в этом проекте обучена на тех же данных и даёт более точную оценку для конкретной выборки, однако на синтетических данных склонна к переобучению.
